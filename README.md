#  Car Rental Platform

A full-stack **Car Rental Marketplace** built with Django that allows users to browse, book, and manage rental vehicles, while hosts can list and monetize their cars. The platform includes role-based dashboards, payment integration, and a notification system.

---

##  Features

###  Authentication & Role Management
- User registration and login
- Role-based access:
  - **Customer** – browse & book cars
  - **Host** – list and manage vehicles
  - **Admin** – approve users and listings

---

###  Vehicle Listings
- Add, edit, and delete car listings
- Categorization (brand, model, type)
- Pricing per day
- Upload vehicle images

---

###  Booking System
- Date-based booking functionality
- Availability management
- Booking confirmation
- Booking history tracking

---

###  Payment Integration
- Integrated payment system (demo/test mode)
- Payment success tracking
- Transaction records

---

###  Reviews & Ratings
- Users can rate vehicles
- Feedback system for better trust and transparency

---

###  Notification System
Users receive notifications for:
- Profile approval
- Listing approval
- Booking confirmation
- Payment success
- Profile updates

---

###  Custom Admin Panel
- Approve/reject user profiles
- Approve listings before publishing
- Manage bookings and transactions
- Image previews in admin dashboard

---

##  Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (default) / PostgreSQL (recommended)
- **Authentication:** Django built-in auth system
- **Media Handling:** Django ImageField
